Coach App - Privacy Policy

Last updated: March 7, 2026

Overview

Coach is a personal endurance coaching application that connects to fitness data services (Whoop, Intervals.icu) to provide personalized training guidance via Telegram. This app is for personal, single-user use only.

Data Collected

The app accesses the following data from connected services:

  • Physiological data: heart rate, HRV, resting heart rate, sleep duration and quality
  • Activity data: workouts, strain scores, training load
  • Recovery data: recovery scores, body metrics

Data Usage

All data is used exclusively for generating personalized coaching recommendations. Data is:

  • Stored locally on the user's own hardware
  • Never shared with third parties
  • Never sold or used for advertising
  • Processed only for coaching analysis

Data Storage

Data is stored in a local SQLite database on the user's machine. No cloud storage is used. The user has full control over their data and can delete it at any time.

Third-Party Services

The app connects to:

  • Whoop API - to retrieve physiological and activity data
  • Intervals.icu API - to retrieve training metrics
  • Anthropic Claude API - to generate coaching analysis (no personal health data is stored by Anthropic)
  • Telegram Bot API - to deliver coaching messages

Contact

For questions about this privacy policy, contact the app developer directly.
